# Basic Syntax

This section covers fundamental Java syntax elements: variables, data types, and operators.

## Variables and Data Types

Java is a statically-typed language, meaning you must declare the type of each variable.

### Primitive Data Types
- `int`: Whole numbers (e.g., 42)
- `double`: Decimal numbers (e.g., 3.14)
- `boolean`: True/false values
- `char`: Single characters (e.g., 'A')

### Variable Declaration
```java
int age = 25;
double price = 19.99;
boolean isStudent = true;
char grade = 'A';
```

## Operators

### Arithmetic Operators
- `+` (addition)
- `-` (subtraction)
- `*` (multiplication)
- `/` (division)
- `%` (modulo/remainder)

### Comparison Operators
- `==` (equal to)
- `!=` (not equal to)
- `<` (less than)
- `>` (greater than)
- `<=` (less than or equal)
- `>=` (greater than or equal)

### Logical Operators
- `&&` (AND)
- `||` (OR)
- `!` (NOT)
